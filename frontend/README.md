# Task Manager - Frontend (React + Vite)

This is the UI layer for the Task Manager, communicating with the Frappe backend via custom API calls.

### Setup Instructions
1. Navigate to this folder: `cd frontend`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`

### Configuration
The `vite.config.js` is set up to proxy requests to `http://127.0.0.1:8000` to avoid CORS issues.
